ROUTE SETUP INSTRUCTIONS

  1. To select an aerodrome, type in the ICAO code (not case sensitive) and press ENTER. Wait for the aerodrome name to appear.

  2. After selecting the arrival aerodrome, the route will intialise. After this point, you must start a new flight to edit the departure or arrival aerodromes.

  3. You can manually add and remove items from the route by using the box provided. Simply select the category of waypoint, and enter the name of the waypoint.

  4. VRPs, navaids and airports are based upon the OpenAIP database, so use the map to access the correct spelling of the name to avoid errors.

  5. Please note that not all cities area available. If a city/town you have chosen is not in the API database you will be notified with an error.
