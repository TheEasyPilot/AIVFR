# AIVFR - AI Assisted Flight Planning Webtool
AIVFR is an AI assisted flight planning webtool for VFR (Visual Flight Rules) pilots in the UK. It gives users the ability to create routes, see weahter reports, create and print flight breifings that include Threat and Error Management, pilot logs, and also configure fuel and mass & balance details. Created by a licensed UK Pilot, this project had the aim of seeing how AI can be integrated into real-world VFR flight planning solutions in the future, as currently they often dominate (and rightfully so) in the more complex, commercial operations.
