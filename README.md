## An Experimental AI assisted Flight Planning Webtool

AIVFR is an experimental AI assisted flight planning webtool for VFR (Visual Flight Rules) pilots in the UK. It gives users the ability to create routes, see weather reports, create and download flight breifings that include Threat and Error Management, pilot logs, and also configure fuel and mass & balance details.

Created by a real licensed pilot, this project had the aim of seeing how AI can be integrated into real-world VFR flight planning solutions in the future, as currently they often dominate (and rightfully so) in the more complex, commercial operations.

This webtool is not adapted for mobile users. For the best experience please use a laptop/tablet.

## Usage

The webtool tries its best to follow the standard flight-planning procedure for a standard VFR cross-country flight. Pilots or sim-pilots with experience will evidently find all the features to be pretty much self explanatory, but to the layperson some general research on how to plan a VFR flight would prove useful.

Information (ⓘ) icons on some pages contain essential information related to the functionality of the specific tool being accessed.

Users who would like to track the development of this webtool should refer to the Notion development page linked at the top of the readme, as well as on the main menu page of AIVFR.

## Debugging routes

`/debug` - View raw JSON data for the current session

`/pdf-template` - Open PDF viewer to see the current state of a PDF before generating
